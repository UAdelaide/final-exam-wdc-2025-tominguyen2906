This folder contains the files for part 2.

## Setup Instructions:

1. Install dependencies:
   ```bash
   npm install
   ```

2. Start a MySQL server

3. Set up the database:
   ```bash
   mysql -u root -p < setup-db.sql
   ```

4. Start the application:
   ```bash
   npm start
   ```

5. Open http://localhost:3000 in your browser
